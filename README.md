###  虛擬網站架設 - YUM YUMZ-好吃的台北城市指南
#### 🧠 專案簡介
- 這是我在《網站企劃 Website Planning》課程中的期末作品。以台北美食為主題，設計一個互動性高、視覺清晰的靜態網頁，模擬真實的美食推薦平台
#### 💡 功能特色
- ✨ 滑鼠懸停評分互動（hover）：讓每道料理的評分動態呈現，提升趣味性
- 🍽 推薦按鈕：模擬推薦機制，提升用戶互動
- 🗺 嵌入 Google 地圖：幫助使用者快速查找美食位置
- 📂 導覽列分類：可切換不同的美食類別，簡潔易用
